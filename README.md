# 539 Kaggle Competition 
- For my implementation I dockerized the application along with the data, the python code is found under scripts. 
- To run the code clone the repo and build the docker image with the following command:
  - docker build -t kaggle-sq .
- Then run the image with the command:
  - docker run --rm -v "$(pwd):/grad-level-term-project-kaggle-competition-Sevren-Quijada" kaggle-sqS
